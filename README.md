# Currency Converter

A simple and responsive Currency Converter web application built using **HTML, CSS, and JavaScript**. This project allows users to convert one currency into another using real-time exchange rates with an easy-to-use interface.

## Features

* Convert between multiple international currencies
* Real-time exchange rate calculation
* User-friendly and responsive design
* Clean and modern UI
* Instant conversion results
* Error handling for invalid inputs

## Technologies Used

* **HTML** for structure
* **CSS** for styling and responsive design
* **JavaScript** for functionality and API integration

## How It Works

Users select the source and target currencies, enter the amount, and click the convert button. The application fetches the latest exchange rate and displays the converted value instantly.

## Purpose of the Project

This project was created to improve frontend development skills and understand API integration using JavaScript while building a practical real-world application.
